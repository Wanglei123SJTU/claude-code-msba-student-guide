# Assignment Requirements (Inferred from `Assg1_MSBA.pdf`)

## Objective
Use `assg1.csv` (30 respondents, 5 attitude items `X1`-`X5`) to perform consumer segmentation using PCA.

## Required Questions to Answer
1. Import, scale, and describe the data in Python.
2. Present the correlation matrix for `X1`-`X5` and plot correlations using `sns.clustermap`; explain exploratory inferences.
3. Perform PCA and report results, including PCA summary and matrix relating factors to original attributes.
4. Write five formal equations linking original attributes to PCA factors.
5. Decide how many factors/components to retain and justify.
6. Propose intuitive names for Factor 1 and Factor 2; explain meaning using strong correlations/equations.
7. Plot respondents on a 2D map using the first two principal components and label axes with factor names.
8. Estimate how many market segments are visible from the map.
9. Describe each segment and link segment characteristics back to original attributes.
10. Identify the segment expected to be most profitable and report its size (%) assuming representativeness.
